# IPL-Performance-Dashboard-Power-BI-Project
This project analyzes IPL ball-by-ball data to evaluate team and player performance using Power BI.

Key Insights:
* Top run scorers and wicket takers
* Strike rate and economy rate analysis
* Boundary percentage by batsmen
* Dismissal type distribution
* Runs vs Strike Rate performance analysis

Tools Used:

* Power BI
* DAX
* Data Modeling
* Interactive Slicers & Visualizations
